### Course concept

Seminar. Recent developments in mathematical imaging focus on improving medical image reconstruction and analysis using variational methods, optimization techniques, and modern machine learning approaches. This seminar discusses current advances in the field and highlights how these methods are used to address challenging imaging problems. Talks should last around 60 minutes, with an additional 15 minutes for questions and discussion.

### Topics (schedule)

1. Volha Turchyna (14.4.) — Learning weakly convex regularizers for convergent image-reconstruction algorithms.
2. Mateusz Marcol (21.4.) — Diffusion at absolute zero: Langevin sampling using successive Moreau envelopes.
3. Benedict Meyer (28.4.) — Uncertainty estimation for learning-based classification of corrupted images.
4. Jaime Díaz-Truechuelo Sanchez-Molini (5.5.) — NeurTV: total variation on the neural domain.
5. Yuliia Pavlenko (12.5.) — Smoothing prior for blind image deblurring.
6. Lilli Schonebeck (19.5.) — Efficient Bayesian computation for low-photon imaging problems.
7. Yirong Qian (2.6.) — Diffeomorphic registration using Sinkhorn divergences.
8. Lars Schmitz (9.6.) — FlowSDF: flow matching for medical image segmentation using distance transforms.
